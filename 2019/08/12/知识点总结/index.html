知识点总结

  1. onClick={this.shareType(‘save’)} 不能携带参数吗？触发无反应 不会触发，因为相当于直接调取函数运行的结果，会直接执行，点击不再反应
  2. transform:translateX(-50%) translateY(-50%); 写在一起和分开写效果不同，why？分开写是相同属性被覆盖，只有最后写的那个属性会生效
  3. webstorm 注册码 ：微信搜索公众号：jetbrains007，进入公众号就可以获得最新激活码
  4. 卸载mac版本的GlobalProtect：
    sudo /Applications/GlobalProtect.app/Contents/Resources/uninstall_gp.sh
  5. React defaultProps, propTypes用法:
    //defaultProps 如果父组件调用子组件的时候不给子组件传值，可以在子组件中使用defaultProps定义的默认值
    //同时propTypes定义父组件给子组件传值的类型,需引入import PropTypes from ‘prop-types’;
  6. PureComponent的作用

这个类的用法很简单，如果你有些组件是纯组件，那么把继承类从 Component 换成 PureComponent 即可。当组件更新时，如果组件的 props 和 state 都没发生改变，render 方法就不会触发，省去 Virtual DOM 的生成和比对过程，达到提升性能的目的。
