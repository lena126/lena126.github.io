chrome常用操作

chroms谷歌浏览器

  1. command+p 能够快速定位指定文件中，实现文件搜索
  2. command+l 跳转到url输入地址栏
  3. command+control+f 当前页面全局或取消全局

1. 所有面板

  • Command-L 光标直接跳至地址栏
  • Control-Tab 转向下一个标签页
  • Control-shift-tab 转向上一个标签页
  • command+↑ ↓ 当前页面顶部/底部
  • Command-N 打开新的窗口
  • Command-W 关闭当前网页，只关闭一个网页
  • command-D 收藏
  • cmd+shift+N 新建一个无痕模式窗口
  • cmd+T 新建一个标签页
  • cmd+opt+L 打开下载内容窗口(只能打开一个)
  • cmd+shift+T 打开下载内容窗口(可打开多个)
  • cmd+shift+B 隐藏/显示 书签栏
  • cmd+opt+B 书签管理页面
  • cmd+shift+delete/x 清除历史数据
  • cmd+ + - 缩放当前页面
  • cmd+R 刷新页面
  • cmd+opt+F 快速必应搜索栏
  • cmd+O 弹出Finder（浏览器页面有效）
  • Cmd+Y 打开历史记录页面

    2. DevTools操作

  • fn + f12 或 cmd+opt+I 调出DevTools
  • cmd+shift+C 调出审查元素窗口
  • cmd+opt+J 打开开发者工具并定位到控制台
  • cmd+opt+C 打开DevTools(不能关闭)
  • Elements审查元素
    • cmd+Z 撤销更改
    • Cmd+Y,Cmd+Shift+Z 重做
    • ↑↓ 上下导航
    • ← → 折叠/展开节点
    • opt+ ← → 折叠/展开所有子节点
    • 双击，enter 修改属性
    • H 隐藏元素
    • fn+F2 编辑As HTML
    • 选中元素右键可以进行：
    • Force State更改元素状态(:active,:hover,:focus,:visited)；
    • scroll into view 定位到页面中该元素
    • 元素上设置断点(更改子元素、更改属性及删除节点)

      3. 样式侧边栏(Style Sidebar)

  • cmd+点击属性 定位到样式属性定义处
  • cmd+点击value 定位到属性值定义处
  • shift+点击颜色块 循环颜色值(rgba,#，hsl等)
  • Tab,Shift+Tab 编辑上/下一个属性
  • ↑↓ 增加/减小值
  • shift+↑↓ 每次以10增加/减小值
  • opt+↑↓ 每次以0.1增加/减小值
  • +添加新的样式选择器

    4. Sources 面板

  • F8,Cmd+\ 暂停/恢复脚本运行
  • F10,Cmd+’ Step over next function call
  • F11,Cmd+; Step into next function call
  • Shift+F11,Cmd+Shift+; Step out of current function
  • Opt+. Select next call frame
  • Opt+, Select previous call frame
  • 点击line number或cmd+B 创建断点
  • line number右键 修改断点状态
  • Cmd+/ 注释
  • Cmd+Opt+S 保存所有修改
  • Cmd+O/P 通过文件名搜索,:number跳转至指定行
  • Opt+W 关闭当前窗口
  • Cmd+D 选择下一个出现位置
  • Cmd+U 撤销上一次操作

    5. Console 控制台

  • Cmd+K,Opt+L 清空控制台
